# Dominion Wellness Solutions - Performance Optimizer

Dominion Wellness Solutions is a premium, data-driven wellness and performance tracking application built with **React Native (Expo)**. It empowers users to monitor their physical capacity, mental resilience, and overall wellness through a sophisticated dashboard and personalized action plans.

## Key Features

### 1. Dynamic Performance Dashboard
- **Radar Chart Header**: Visualizes five core performance pillars (Physical, Mental, Cohesion, Purpose, Recovery).
- **Optimal Performance Score (OPS)**: A real-time summary of user health with trend tracking.
- **Metric Cards**: Detailed breakdown of individual performance drivers with color-coded status badges.

### 2. Daily Assessment Flow
- **3-Step Check-in**: Interactive screens for assessing energy, focus, stress, and motivation.
- **Lifestyle Tracking**: Guided logs for sleep quality, physical activity, and nutritional habits.
- **Streak Tracking**: Rewarding feedback loop with "Reflection Streaks" to encourage consistency.

### 3. Comprehensive Performance Reports
- **Weekly Progress Charts**: SVG-based area charts visualizing 7-day performance trends.
- **Behavioral Grids**: Heatmap-style grids for Sleep, Stress, and Activity patterns.
- **Actionable Insights**: AI-ready personalized and leader action plans with structured task cards.

### 4. Account & Profile Management
- **Profile Customization**: Detailed editing of personal, organizational, and physical metrics.
- **Secure Authentication**: Support for password changes and secure logout/delete account flows with high-fidelity confirmation modals.
- **Help & Support**: Integrated Help Center and Contact Support forms.

## Tech Stack

- **Framework**: [Expo](https://expo.dev/) (React Native)
- **Language**: TypeScript
- **Icons**: Lucide React Native
- **Data Visualization**: React Native SVG (Custom Radar, Area, and Bar charts)
- **Styling**: Vanilla React Native StyleSheet (Premium "Green Market" Aesthetic)
- **Typography**: Manrope (Google Fonts)
- **Navigation**: Expo Router (File-based routing)
